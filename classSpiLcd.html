Detailed Description

SPI Attached LCD.

This is an adaptation of the Arduino LiquidCrystal library for the NHD-0420DZW-AY5-ND lcd display, made by NewHaven. The display should be HD44780 compatible but isn't. Differences are some of the control commands (cursor on/off), language setting and especially initialization sequence.
