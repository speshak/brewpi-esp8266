\doxysubsection{Detailed Description}
Ticks -\/ interface to a millisecond timer

With more code space, Ticks would have been a virtual base class, so all implementations can easily provide the same interface. Here, the different implementations have no common (virtual) base class to save code space. Instead, a typedef is used to compile-\/time select the implementation to use. If that implementation doesn\textquotesingle{}t implement the Ticks interface as expected, it will fail to compile. A Ticks implementation that increments the millis count each time it is called. This is used for testing.
