\doxysubsection{Detailed Description}
General interface for all temperature sensor devices.

Wraps up a \mbox{\hyperlink{classBasicTempSensor}{Basic\+Temp\+Sensor}} and processes the values through several filters. The filters are either \mbox{\hyperlink{classCascadedFilter}{Cascaded\+Filter}} or \mbox{\hyperlink{classFixedFilter}{Fixed\+Filter}} implementations, depending on the compile time value of the T\+E\+M\+P\+\_\+\+S\+E\+N\+S\+O\+R\+\_\+\+C\+A\+S\+C\+A\+D\+E\+D\+\_\+\+F\+I\+L\+T\+ER macro.
