Detailed Description

Defines global config for the brewpi project.

This file is included in every file in the project to ensure conditional compilation directives are recognized.

ConfigDefault.h contains the default settings, and produces a standard Hex file. To customize the build, users may add settings to Config.h, or define symbols in the project.
